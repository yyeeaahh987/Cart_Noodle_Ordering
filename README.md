This is a e-commence project built with MERN.

# Step 0

Use Terminal / Powershell in the root directory
Use below commands to install library

### backend part

> cd backend
> npm i

### frontend part

> cd client
> npm i

# Step 1

Use Terminal / Powershell in the root directory

### backend part

> cd backend
> npx nodemon

### frontend part

> cd client
> npm start
> Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
